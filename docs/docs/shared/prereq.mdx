<details>
  <summary>Installation Instructions</summary>

  These installation instructions may not be complete and may vary from system to system. These instructions should work on MacOS and Linux. If you are a windows user please open a pull request and we will update these instructions, Godspeed brave soldier.

  ```bash
  # Install Rustup
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  # Install Rust 1.81.0
  rustup install 1.81.0
  # Install Solana using solana-install (here installing 2.1.13)
  sh -c "$(curl -sSfL https://release.anza.xyz/v2.1.13/install)"
  # Install Anchor avm
  cargo install --git https://github.com/project-serum/anchor avm --locked
  # Install Anchor 0.30.1 (only for POW POW demo, otherwise not needed)
  avm install 0.30.1
  # Install Bonsol CLI, Risc0 Toolchain, Flatbuffers
  curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/anagrambuild/bonsol/refs/heads/main/bin/install.sh | sh
  # Install Flatbuffers
  # follow the instructions at https://flatbuffers.dev/building/
  # or if on a mac use brew
  brew install flatbuffers@24.3.25

  # Install Docker
  # For WSL users: We recommend installing Docker directly in WSL instead of Docker Desktop
  # See detailed guide at: /how-to-guides/setup/docker-wsl

  # For Ubuntu/Debian (including WSL Ubuntu):
  sudo apt-get update
  sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
  sudo usermod -aG docker $USER  # Allow docker commands without sudo
  ```
</details>

Don't forget to install the required dependencies for your system.
